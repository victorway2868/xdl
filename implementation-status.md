## 🚀 如何测试当前功能

### 1. 启动应用
```bash
npm install
npm run start
```

### 2. 测试弹幕功能
1. 在主页点击 "💬 弹幕助手" 按钮
2. 点击右上角设置按钮
3. 输入任意数字作为房间号（如：123456）
4. 点击连接按钮
5. 观察模拟弹幕数据的实时显示

### 3. 测试场景编辑器
1. 在主页点击 "📺 场景编辑器" 按钮
2. 如果已在弹幕助手中连接，会显示直播间数据
3. 如果未连接，会提示前往弹幕助手连接

### 4. 测试数据共享
1. 在弹幕助手中连接直播间
2. 切换到场景编辑器页面
3. 观察直播间数据是否同步显示
4. 返回弹幕助手，数据应保持连接状态
